# Mediumclone

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 15.2.8.

# Descrizione:
Client Medium Clone, con sistema di registrazione e auth degli utenti, user profile, feetching feeds da API REST, e possibilità di creare e gestire i propri articoli o post.

# Framework usati:
  Angular version 15.2.8
# Libraries 
  Rxjs
  Ngrx (con pattern Redux per la gestione dello stato globale dell'applicazione, https://ngrx.io/)
  Tailwind per il CSS e responsive
  Query-string (https://www.npmjs.com/package/query-string)
